      • alwaysSpan

        public BidiFormatter.Builder alwaysSpan(boolean alwaysSpan)
        Specifies whether the BidiFormatter.spanWrap(java.lang.String, boolean, boolean) and BidiFormatter.spanWrapWithKnownDir(com.google.template.soy.data.Dir, java.lang.String, boolean, boolean) methods of the BidiFormatter to be built should produce a stable span structure, i.e. wrap the string in a span even when its directionality does not need to be declared. The default is false.
