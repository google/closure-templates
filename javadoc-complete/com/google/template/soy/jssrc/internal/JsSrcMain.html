        Generates JS source files given a Soy parse tree, an options object, an optional bundle of translated messages, and information on where to put the output files.
        Parameters:
        soyTree - The Soy parse tree to generate JS source code for.
        jsSrcOptions - The compilation options relevant to this backend.
        locale - The current locale that we're generating JS for, or null if not applicable.
        msgBundle - The bundle of translated messages, or null to use the messages from the Soy source.
        outputPathFormat - The format string defining how to build the output file path corresponding to an input file path.
        inputPathsPrefix - The input path prefix, or empty string if none.
        Throws:
        SoySyntaxException - If a syntax error is found.
        IOException - If there is an error in opening/writing an output JS file.
