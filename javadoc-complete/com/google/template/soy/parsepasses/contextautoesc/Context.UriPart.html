    public static enum Context.UriPart
extends Enum<Context.UriPart>
    Describes the part of a URI reference that the context point is in.

    We need to distinguish these so that we can

    • normalize well-formed URIs that appear before the query,
    • encode raw values interpolated as query parameters or keys,
    • filter out values that specify a scheme like javascript:.
