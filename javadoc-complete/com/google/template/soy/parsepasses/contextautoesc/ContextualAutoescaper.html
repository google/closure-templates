Class ContextualAutoescaper

java.lang.Object
  com.google.template.soy.parsepasses.contextautoesc.ContextualAutoescaper

public final class ContextualAutoescaper
extends Object

Inserts directives into print commands by looking at the context in which a print appears, and derives templates and rewrites calls so that each template is entered only in contexts consistent with its escaping conventions. E.g. it will rewrite

{template example autoescape="contextual"} <p>Hello, {$world}!</p> {/template} to {template example autoescape="contextual"} <p>Hello, {$world |escapeHtml}!</p> {/template}

Parameters:
