    public class MsgNode
extends AbstractBlockCommandNode
implements SoyNode.ExprHolderNode, SoyNode.MsgBlockNode
    Node representing a 'msg' block. Every child must be a RawTextNode, MsgPlaceholderNode, MsgPluralNode, or MsgSelectNode.

    Important: Do not use outside of Soy code (treat as superpackage-private).

      • getAndRemoveGenderExprs

        @Nullable
public List<ExprRootNode<?>> getAndRemoveGenderExprs()
        Returns the list of expressions for gender values and sets that field to null. Note that this node's command text will still contain the substring genders="...". We think this is okay since the command text is only used for reporting errors (in fact, it might be good as a reminder of how the msg was originally written).
