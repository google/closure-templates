public interface SoyJavaRuntimePrintDirective
extends SoyPrintDirective

Interface for a Soy print directive implemented for Java runtime rendering. The apply() method of this interface should be the same as the applyForTofu() method of interface SoyTofuPrintDirective. The difference is that directives implementing this interface may be useful to the Soy compiler during optimization passes.

Important: This may only be used in implementing print directive plugins.
