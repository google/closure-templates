Interface SoyJavaPrintDirective

    public interface SoyJavaPrintDirective
extends SoyPrintDirective
    Interface for a Soy print directive implemented for Java runtime rendering. Directives implementing this interface will be used during Tofu rendering, and may also be used during optimization passes if the directive is also marked with annotation @SoyPurePrintDirective.

    Important: This may only be used in implementing print directive plugins.
