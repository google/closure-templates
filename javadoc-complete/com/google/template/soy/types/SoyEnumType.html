com.google.template.soy.types

Interface SoyEnumType

  • All Superinterfaces:
    SoyType

    public interface SoyEnumType
extends SoyType
    Type representing an enumeration. Enum types have a unique name, and can have zero or more enum constant values. Enum values are always integers.

    Enum types are always referred to by their fully-qualified name.

    Important: Do not use outside of Soy code (treat as superpackage-private).
