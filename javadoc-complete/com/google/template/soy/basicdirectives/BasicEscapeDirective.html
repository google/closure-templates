  • java.lang.Object

    public abstract class BasicEscapeDirective
extends Object
implements SoyJavaPrintDirective, SoyJsSrcPrintDirective
    An escaping directive that is backed by Sanitizers in java, and soyutils.js or the closure equivalent in JavaScript. See GenerateSoyUtilsEscapingDirectiveCode which creates the JS code that backs escaping directives.
