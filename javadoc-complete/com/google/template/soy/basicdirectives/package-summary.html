Package com.google.template.soy.basicdirectives

Plugin package for basic print directives.

See: Description

  • Class Summary 
    Class Description
    BasicDirectivesModule
    Guice module for basic Soy print directives.
    BasicEscapeDirective
    An escaping directive that is backed by Sanitizers in java, and soyutils.js or the closure equivalent in JavaScript.
    ChangeNewlineToBrDirective
    A directive that replaces newlines (\n, \r, or \r\n) with HTML line breaks (<br>).
    CleanHtmlDirective
    Implements the |cleanHtml directive, which removes all but a small, safe subset of HTML from its input.
    FilterImageDataUriDirective
    Implements the |filterImageDataUri directive, which only accepts data URI's corresponding to images.
    InsertWordBreaksDirective
    A directive that inserts word breaks as necessary.
    TextDirective
    Internal-only directive indicating content is to be treated as plain text.
    TruncateDirective
    A directive that truncates a string to a maximum length if it is too long, optionally adding ellipsis.
