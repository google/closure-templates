Class IndentedLinesBuilder

public class IndentedLinesBuilder
extends Object
implements CharSequence, Appendable

A wrapped StringBuilder used for building text with indented lines.

Important: Do not use outside of Soy code (treat as superpackage-private).

The maximum indent length is 24.
