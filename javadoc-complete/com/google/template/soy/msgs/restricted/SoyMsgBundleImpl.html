        public SoyMsgBundleImpl(@Nullable
                String localeString,

        Note: If there exist duplicate message ids in the msgs list, the first one wins. However, the source paths from subsequent duplicates will be added to the source paths for the message.
        Parameters:
        localeString - The language/locale string of this bundle of messages, or null if unknown. Should only be null for bundles newly extracted from source files. Should always be set for bundles parsed from message files/resources.
        msgs - The list of messages. List order will become the iteration order.
