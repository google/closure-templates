public class SoyMsgPluralCaseSpec
extends Object

Represents a plural case value. A plural case value can be either a number, or one of ZERO, ONE, TWO, FEW, MANY or OTHER. Here, a number is represented by the number explicitValue with status set to EXPLICIT and the remaining by an enum value.
