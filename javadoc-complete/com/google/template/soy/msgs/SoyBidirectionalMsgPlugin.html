Interface SoyBidirectionalMsgPlugin

  • All Superinterfaces:
    SoyMsgPlugin

    public interface SoyBidirectionalMsgPlugin
extends SoyMsgPlugin
    Plugin for implementing a specific message file format. Supports postprocessing of msgs files by having methods to both generate and parse both extracted and translated msgs files.
