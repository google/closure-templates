    public interface ExprNode
extends Node
    This class defines the base interface for a node in the Soy expression parse tree, as well as a number of subinterfaces that extend the base interface in various aspects. Every concrete node implements some subset of these interfaces. The top level definition is the base ExprNode interface.

    Important: Do not use outside of Soy code (treat as superpackage-private).
